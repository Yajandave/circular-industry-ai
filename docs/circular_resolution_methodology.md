# Circular Resolution Methodology

Circular Industry AI uses the rules engine as the locked decision source. The Circular Resolution Engine does not change the recommendation. It adds a practical circular economy intervention plan around it.

## Method logic

1. Read the industrial stream and locked recommendation.
2. Select a material-specific circular playbook.
3. Frame the circular economy problem.
4. Generate a specific resolution idea.
5. Add supplier/procurement action.
6. Add process redesign action.
7. Add industrial symbiosis screening where relevant.
8. Define a pilot plan and KPIs.
9. Add evidence requirements and decision gates.
10. Add a claim boundary and fallback route.

## Circular economy distinction

Waste management asks where a waste stream should go. Circular economy asks why the stream exists, how it can be prevented, whether value can be retained at a higher level, and what evidence is needed before a claim can be made.

## Control rule

Rules decide. Resolution plans design the intervention. Humans approve implementation and claims.
