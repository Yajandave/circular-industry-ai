# Product Language Guardrails

Circular Industry AI should be described as an industry / market / professional-grade circular economy, ESG, EIA and sustainability intelligence dashboard.

## Preferred language

Use:

- professional product
- professional system
- intelligence dashboard
- operator-facing dashboard
- decision-support platform
- circular economy intelligence
- sustainability intelligence
- ESG/EIA-aligned risk and evidence screening
- evidence-controlled workflow
- claim-safe review workflow
- rules-locked decision record
- operator drilldown
- decision triage
- supplier-loop intelligence
- circular procurement intelligence

## Avoid

Avoid using these as the main framing:

- portfolio project
- demo
- showcase
- recruiter-ready
- screenshot-ready
- project polish
- toy app
- proof-of-concept

## Acceptable context

It is acceptable to say the repository demonstrates professional capability, but the product itself should not be described as a portfolio project.

Better:

> This system demonstrates professional circular economy and sustainability intelligence workflows.

Avoid:

> This is a portfolio project for job applications.

## Governance wording

Always preserve this boundary:

> The rules engine remains the locked decision source. AI/LLM may explain, summarise and draft, but it cannot override risk level, review status, rule applied, claim boundary, evidence controls, legal/compliance status or verified impact.
